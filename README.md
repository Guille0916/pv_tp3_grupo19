# pv_tp3_grupo19

Aplicación web desarrollada con React para gestionar proyectos educativos. El proyecto funciona como una Single Page Application usando React Router, con vistas para Dashboard, listado de proyectos, detalle dinámico por URL y perfil de usuario. La interfaz utiliza Bootstrap para mejorar formularios, botones, tarjetas y mensajes.
